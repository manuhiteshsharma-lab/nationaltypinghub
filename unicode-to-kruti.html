Why Convert Between Mangal & Kruti Dev?

While Mangal (Unicode) is the standard font for the internet, WhatsApp, and modern software, many Indian government departments still rely on legacy formats for official documentation. If you are preparing drafts, letters, or data entry work for state exams like UPSSSC, Rajasthan LDC, or Bihar Beltron, you will often be required to submit your work in Kruti Dev 010 or Devlys font.

Because Kruti Dev is a non-Unicode font based on the traditional Remington typewriter layout, pasting standard web Hindi directly into an official Word document will result in unreadable text. Our bidirectional converter solves this problem instantly by rewriting the formatting in whichever direction you need.

No Need to Learn the Remington Layout

If you only know how to type in Mangal (Inscript) or using Google Input Tools (English to Hindi), there is no need to learn the complex Kruti Dev or Devlys keyboard layout! You can type your documents normally in Mangal, paste the text into our tool, and generate the required legacy font.
