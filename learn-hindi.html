Chapter 1: Choosing Your Font

Mangal vs. Kruti Dev

The first step in learning Hindi typing is knowing which keyboard layout your specific exam requires. Unlike English, Hindi has two entirely different mapping systems.

🔵 Mangal (Inscript)

  • Used For: SSC, Railways, CPCT, and most modern Central Government exams.
  • How it works: It uses Unicode. The layout is phonetic (vowels are on the left hand, consonants are on the right).
  • Pro: Once learned, it works natively on any modern computer or website.

🟠 Kruti Dev 010 (Remington)

  • Used For: UPSSSC, Beltron, Rajasthan State Exams, and traditional clerical roles.
  • How it works: It mimics an old mechanical typewriter. It maps Hindi symbols directly onto English keystrokes.
  • Pro: If you know English typing, the finger stretches feel very similar.

Chapter 3: Matras & Half Letters

Mastering the Complexities

Hindi typing is significantly harder than English because of half-letters and vowel markers (Matras). Here is how you master them based on your layout.

In Mangal (Inscript): The Halant (्)

  • Inscript is highly logical. To make a half-letter, you simply type the full letter, and then add a Halant (्) using the D key.
  • Example: To type क्या, you type क (k) + ् (d) + य (/) + ा (e).
  • Matras are usually typed after the consonant. Example: To type कि, type क (k) then ि (f).

In Kruti Dev: The Typewriter Method

  • Kruti Dev uses the Shift key heavily to access pre-made half-letters. Holding Shift + V types a half ट (V).
  • If there is no specific key for a half-letter, you type the full letter and use the ~ (Tilde) key to add the Halant.
  • Crucial Difference: The small "i" matra (ि) must be typed before the consonant. To type कि, you press f then k.
